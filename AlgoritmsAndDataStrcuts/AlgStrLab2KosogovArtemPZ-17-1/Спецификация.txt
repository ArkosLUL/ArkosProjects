 Общая (с обработкой исключительных ситуаций) реализация неограниченных     
                              стеков строк	                   					
     (представление - объект (object))                                      
============================================================================
                           СПЕЦИФИКАЦИЯ                                     
                        абстракции данных                                   
                   "Неограниченные стеки обобщенного класса Т Stack"                           
                                                                            
   Stack = DATA TYPE IS Stack, Push, Pop, Count,  
            IsEmpty, Copy, Concatenate, Divide, foreach, DeleteStack                  
                                                                            
   ОПИСАНИЕ                                                                 
                                                                            
      Стеки обобщенного класса T Stack - это неограниченные стеки             
         обобщенного класса T.(При реализации структуры, вместо класса Т возможно использование как String, int так и пользовательских классов(Person,Car),
	 при чем без необходимости внесения изменений в струкутуру данных)
	 Стеки ИЗМЕНЯЕМЫЕ: операции push и pop   
         добавляют и удаляют строки из Stack                      
                                                                            
   ОПЕРАЦИИ                                                                 
                                                                            
      public Stack()  //конструктор                                                 
         EFFECTS Инициализирует новую, пустой стек Stack             
                                                                            
      public void Push(T Tobj)                                             
         MODIFIES this                                                      
         EFFECTS  Добавляет Tobj на вершину стека                            
                                                                            
     public T Pop()                                               
         MODIFIES this                                                      
         EFFECTS ЕСЛИ not IsEmpty ТО                                        
                    Удалаяет элемент из вершины стека И сигнализирует Ok   
                 ИНАЧЕ                                                      
                    Сигнализирует Underflow                                 
                 ВСЕ-ЕСЛИ                                                   
            
     public T Peek()
	  EFFECTS ЕСЛИ not IsEmpty ТО                                        
                    Возвращает копию из вершины стека, не удаляя его из стека 
                 ИНАЧЕ                                                      
                    возвращает null                                 
                 ВСЕ-ЕСЛИ
                                                                                                                                                                                                                                                                                                                                                            
     public int Count()                                                  
         EFFECTS Возвращает число элементов в стеке                               
                                                                                                      
     public Stack<T> Copy()                                            
         EFFECTS Возвращает новый стек Stack<T>, содержащий те же        
            элементы, что и this. 
      
     public Concatenate(Stack<T> secondHalf)                                         
         EFFECTS Помещает все элементы стека-параметра в вызывающий this стек Stack<T> очередь UnlimQueue<T>,
		 в результате this содержит элементы обоих стеков.

     public Stack<T> Divide(int elemNumber): Signals
	  EFFECTS Если this.IsEmpty 
			Сигнализирует Underflow
		  Все-Если
		  Если elemNumber >= this.Counter
			Сигнализирует Underflow
			Иначе 
				Возвращает новый стек Stack<T>, содержащий количество элементов равное elemNumber, из this стека при этом
				достается такое же количество элементов
				Все-Если
		 Все-Если

      public  DeleteStack()
            EFFECTS Переставляет указатель на голову (Head) вызывающего (this) стека на null
		    Изменяет Counter на 0.	  
	


	                                          
                                                                            
----------------------------------------------------------------------------
	foreach = GENERATOR TYPE IS public IEnumerator<T> GetEnumerator()           
                                                                            
       ОПИСАНИЕ                                                             
                                                                            
       Итератор foreach используются для выполнения итераций по элементам   
          очередей обобщенного класса Т Stack. Элементы стека выдаются 
          последовательно в порядке от вершины (Head) ко дну стека через
	  указатель (Next).
	  Реализация генератора зависит от представления стеков  
          обобщенного класса Т Stack                                             
                                                                            
	ОПЕРАЦИИ                                                                
                                                                            
	public IEnumerator<T> GetEnumerator()
		Effects Реализует интерфей IEnumerable<T>,который позволяет использовать итератор foreach(особенность языка С#),
			для прохода по стеку                                                 
----------------------------------------------------------------------------
                                                                            
	                                
   END  Stack                                                              
============================================================================
	Используя  итератор foreach задачу вывод в консоль 
	элементов обобщеного стека Stack можно реализовать следующим образом:           
                                                                            
	foreach (var elem in MyStack)
               Console.WriteLine(elem.ToString());
