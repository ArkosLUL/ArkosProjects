  ОСОБЕННОСТИ РЕАЛИЗАЦИИ:
    - Программа написана на языке C#; 
    - Среда программирования: Microsoft Visual Studio 2012

    - Работа с текущим каталогом.
    - Работа с указанным каталогом.
    - Работа со всем фрагментом файловой структуры.
    - Работа со всеми файлами каталога.
    - Поддерживается переменное число параметров командной строки.
    - Поддерживается режим подсказки.
    - Поддерживается режим умолчания параметров утилиты.
    - Переменная окружения используется для задания режимов умолчания.
    - Программа формирует коды завершения:
        0 - нормальное завершение
        1 - выход по EXIT
        2 - неправильное выражение
        3 - выход по параметру '/?'  (режим справки)
        4 - неверный атрибут
        5 - аргумент не соответствует ни одному из параметров
        6 - каталог (путь) не существует
    - Атрибуты файлов учитываются при обработке.
    - Ключевые параметры используются.
    - Поддерживается дружественная обработка ошибок.
